# 🎓 Student Management System

A Student Management System built using Python and Streamlit that allows users to manage students and teachers through a simple graphical interface.

The project was built to practice Object-Oriented Programming concepts such as inheritance, abstraction, encapsulation, and polymorphism while also learning JSON data handling and Streamlit deployment.

---

## ✨ Features

- 👨‍🎓 Register Students
- 👨‍🏫 Register Teachers
- 📝 Add Grades
- 📊 View Student Details
- 📋 View Teacher Details
- 💾 JSON Data Storage
- 🖥 Streamlit User Interface

---

## 🛠 Built With

- Python
- Streamlit
- JSON
- Object-Oriented Programming

---

## 🚀 Installation

```bash
git clone https://github.com/YourUsername/student-management-system.git

cd student-management-system

pip install -r requirements.txt

python -m streamlit run app.py
```
