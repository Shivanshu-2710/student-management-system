# 🎓 Student Management System

A simple Student Management System built using Python OOP and Streamlit.

- Add Student
- Add Teacher
- Add Grades
- View Student Details
- View Teacher Details

## Technologies Used

- Python
- Streamlit

## How to Run

```bash
pip install -r requirements.txt
